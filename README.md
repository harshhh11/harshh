# Event Ease 🎉

Event Ease is a modern web application built to simplify campus event discovery and management.  
It helps students explore events, stay updated, and engage with campus activities through a clean and interactive interface.

---

## 🚀 Features

- User authentication (Email & Google login)
- Browse and search campus events
- View detailed event information
- Personalized user dashboard
- AI-powered event assistant
- Campus heatmap for event activity
- Responsive and accessible UI

---

## 🛠️ Tech Stack

This project is built using:

- **React** (with TypeScript)
- **Vite**
- **Tailwind CSS**
- **shadcn/ui**
- **Supabase** (Authentication & backend services)
- **React Router**
- **TanStack Query**

---

## 📦 Getting Started

### Prerequisites

Make sure you have the following installed:

- **Node.js** (v18 or higher recommended)
- **npm**

### Local Setup

```sh
# Clone the repository
git clone https://github.com/harshhh11/web-canvas.git

# Navigate to the project directory
cd web-canvas

# Install dependencies
npm install

# Start the development server
npm run dev
